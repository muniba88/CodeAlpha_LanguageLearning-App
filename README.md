#  EduLingo - Language Learning App

EduLingo is a modern Flutter-based Language Learning App designed to help users learn multiple languages through interactive lessons, quizzes, flashcards, and progress tracking.

The app provides a clean and user-friendly interface with personalized learning features and Supabase backend integration for authentication and data storage.

---

# ✨ Features

## 🔐 Authentication
- User Signup
- User Login
- Logout Functionality
- Secure authentication using Supabase

## 📚 Language Learning
- Choose preferred language
- Learn vocabulary and phrases
- Interactive onboarding flow
- Flashcard-style learning

## 📊 Progress Tracking
- Daily learning progress
- Activity tracking
- Achievements section
- Selected language cards

## 👤 Profile Management
- User profile section
- Display user information
- Add multiple learning languages
- Store user language preferences

## 🎨 Modern UI
- Clean and responsive design
- Smooth onboarding screens
- Modern green educational theme
- Responsive layouts for different screen sizes
